# jQuery assignments

Create an webpage that links to the cdn of jQuery

## 1 Event

Create a button and a paragaph with the text of "No value".

When the button is pressed, change the textContent of the paragaph to "Hello World!".

## 2 List

Create a text input. Make an unordered list.
When the previous button is clicked, add a new list item in the list with the text content of the input field.

## 3 CSS

Make a list with 8 items. Fill them with whatever you like, ex lorem ipsum.

Make the list look a little nicer with jQuery css, not a stylesheet.
